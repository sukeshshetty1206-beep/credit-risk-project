# Credit Risk Assessment using PCA and WKNN

## Objective
The objective of this project is to predict the probability of loan default using machine learning techniques.

## Dataset
A sample loan dataset containing:
- Loan amount
- Interest rate
- Annual income
- Debt-to-income ratio
- Loan status

## Methodology
1. Data preprocessing
2. Feature conversion
3. PCA (Principal Component Analysis) for dimensionality reduction
4. WKNN (Weighted K-Nearest Neighbors) for classification

## Results
- Achieved high accuracy
- Generated classification report
- Visualized loan default distribution

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Conclusion
The model successfully predicts loan default using PCA and WKNN.
